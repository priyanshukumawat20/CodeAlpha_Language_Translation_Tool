# CodeAlpha_Language_Translation_Tool
A language translation tool built using Python, Gradio, and Googletrans

## Task 1 - Artificial Intelligence Internship at CodeAlpha

A simple and interactive **Language Translation Tool** built using Python.

## 🔧 Tech Stack
- Python
- Googletrans (Google Translate API)
- Gradio (for UI)

## ✨ Features
- Enter any text and translate it instantly
- Select source and target language from dropdown
- Clean and simple user interface

## 🚀 How to Run
1. Install dependencies: `pip install googletrans==4.0.0-rc1 gradio`
2. Run the notebook in Google Colab or Jupyter
3. Use the Gradio link to access the UI
